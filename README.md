## XJSwiftKit

集成常用的第三方库，封装了多种基类，为Foundation和UIKit提供多种分类方法，使用Swift快速开发一款应用。    

### BaseViewController
```
  // Base
  XJBaseViewController
  // Navigation
  XJNavigationViewController
  // TabBar
  XJTabBarViewController
  // WebView
  XJWKWebViewController
  // Segment  
  XJSegmentViewController
  // Search
  XJSearchViewController
```
